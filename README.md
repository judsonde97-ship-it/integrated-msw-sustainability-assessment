# Integrated Sustainability Assessment of Municipal Solid Waste Management Systems in Marrakech

Material Flow Analysis (STAN) and Cost Efficiency Evaluation

---

## Project Overview

This project evaluates alternative municipal solid waste management systems in Marrakech through Material Flow Analysis (MFA) and Cost Efficiency Analysis.

The study compares three scenarios and assesses:

- Waste Flows
- Resource Recovery
- Landfill Diversion
- Circular Economy Performance
- Cost Efficiency

---

## Methods

- Material Flow Analysis (STAN)
- Scenario Modelling
- Resource Recovery Assessment
- Cost Efficiency Analysis

---

## Key Findings

✔ 33% landfill diversion achieved under Scenario III

✔ Increased resource recovery through composting

✔ Circular Economy-oriented waste management pathways

✔ Economic trade-offs identified
